# Prompt Engineering Lab: Python Chatbot

This project demonstrates how different prompting techniques—**Zero-Shot**, **Persona/Context**, and **Constraint Formatting**—affect the quality and structure of AI-generated code.

## The Tasks

### Task 1: Zero-Shot
- **Prompt:** "Write a Python chatbot."
- **Observation:** The code was functional but lacked structure and specific features.

### Task 2: Persona + Context
- **Prompt:** Senior Dev persona + Student context.
- **Observation:** The code became more user-friendly and included personalized greetings.

### Task 3: Final Polished Version
- **Prompt:** Added functional programming requirements and line-by-line commenting.
- **Result:** See `main.py`. This version is modular, clean, and highly readable.
